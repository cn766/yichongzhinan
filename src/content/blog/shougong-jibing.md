## 守宫会生病——你要做的不是"吓死"，而是"知道怎么办"

守宫（豹纹守宫、睫角守宫等）是宠物爬宠里**抗病能力最强**的种类之一。但"最强"不等于"不会生病"。

事实上，大多数守宫的疾病，都是**饲养不当引起的**——温度不对、湿度不对、垫材有问题、食物不干净。换句话说：**你就是你守宫的医生，预防比治疗重要一万倍。**

本文会教你识别守宫最常见的6种疾病，以及——**什么时候可以在家处理，什么时候必须立刻冲到异宠兽医那里去。**

## 一、脱皮不全（Retained Shed / Dysecdysis）

### 症状

- 眼睛上有一层浑浊的"白膜"（守宫的"眼镜"没有蜕掉）
- 脚趾/尾巴尖有残留的旧皮（看起来像"袜子"或"手套"）
- 全身或局部有透明度不同的旧皮残留

**严重程度判断**：
- 轻度：只有眼睛或脚趾有几片残留 → 可以在家处理
- 中度：身体大面积残留，或眼睛浑浊超过3天 → 建议看兽医
- 重度：眼睛浑浊超过1周，或脚趾残留导致缺血性坏死 → **立即看兽医**

### 原因

- **湿度太低**——这是#1原因。守宫需要50-60%湿度才能正常蜕皮
- 没有"摩擦表面"——守宫需要粗糙表面来蹭掉旧皮
- 营养不良——维生素A缺乏会影响皮肤健康
- 压力——新环境、频繁上手等压力会干扰蜕皮周期

### 家庭处理方法（轻度案例）

**步骤1：增加湿度**
- 在饲养箱里放一个"蜕皮盒"——打孔的塑料盒，里面放湿润的苔藓或厨房纸
- 或者每天2-3次用喷雾器喷饲养箱壁（不要直接喷守宫）

**步骤2：温水浸泡**
- 用一个浅塑料盆，装32-35°C温水（深度不超过守宫肩膀）
- 把守宫放进去浸泡15-20分钟
- 用湿棉签或湿纱布，**轻轻**擦拭残留的旧皮
- **关键**：如果旧皮不松动，不要强行撕——会撕掉新皮，导致感染和出血

**步骤3：人工辅助**
- 浸泡后，用湿棉签轻轻滚动擦拭眼睛周围的残留皮
- 如果眼睛上的"眼镜"还是不脱落，**停止**——强行操作会伤到眼球
- 此时应该看兽医，他们会用专业工具和专业麻醉下操作

### 预防

- 维持湿度50-60%（用湿度计监测）
- 提供粗糙的石头或树皮（守宫可以蹭皮）
- 蜕皮期不要上手打扰
- 蜕皮前1-2天，守宫会变得更白/更不透明——这是信号，提前准备蜕皮盒

## 二、肠胃炎（Gastroenteritis / "湿尾"）

### 症状

- 腹泻或"湿尾"——便便不成形，沾在肛门周围
- 便便有异味（正常守宫便便几乎无臭味）
- 便便中有未消化的食物
- 守宫活动量下降、拒食
- 脱水迹象（皮肤弹性差、眼睛凹陷）

### 原因

- **细菌感染**（沙门氏菌、大肠杆菌等）——最常见
- 食物不干净（喂了被污染的昆虫）
- 垫材问题（松木屑的油脂会刺激肠道）
- 温度过低（免疫力下降）

### 家庭处理 vs 看兽医

| 症状严重程度 | 处理方式 |
|---------------|----------|
| 轻度腹泻（1-2天，守宫还活跃） | 提高温度至29-30°C，停食2-3天观察 |
| 中度（腹泻3-5天，守宫活动下降） | **看兽医**——需要粪检和可能的抗生素 |
| 重度（腹泻>5天，拒食，脱水） | **立即看兽医**——可能已脱水，需要输液 |

**警告**：肠胃炎可以在48-72小时内杀死一条守宫。如果你不确定严重程度，**看兽医**。

### 兽医会做什么？

1. 粪检——确定是细菌/寄生虫/病毒
2. 可能做血检——评估脱水程度和器官功能
3. 开抗生素（通常是恩诺沙星/Enrofoxacin）
4. 严重脱水时——皮下或静脉输液

### 预防

- 食物来源可靠（不要喂野外抓的昆虫）
- 垫材不用松木屑
- 维持正确温度（守宫免疫系统需要28-30°C）
- 饮水清洁（每周换水，水盆定期刷洗）

## 三、呼吸道感染（Respiratory Infection / RI）

### 症状

- 嘴巴张开呼吸（"gaping"）
- 呼吸时有"咔嚓"声或喘息声
- 鼻孔有气泡或黏液
- 活动量明显下降
- 拒绝进食
- 眼睛半闭，精神萎靡

### 原因

- **温度过低**——最常见原因。守宫需要热点28-30°C
- 湿度过高+通风不良——细菌/真菌滋生
- 压力——新环境、运输、混养等

### 家庭处理 vs 看兽医

| 症状 | 处理方式 |
|------|----------|
| 轻微：偶尔张嘴，但还活跃、还进食 | 提高温度至29-30°C，增加通风，观察48小时 |
| 中度：频繁张嘴，活动下降，拒食1-2天 | **看兽医** |
| 重度：嘴巴有泡沫，呼吸有声音，完全拒食 | **立即看兽医**——可能已发展为肺炎 |

**警告**：RI可以在3-5天内发展为肺炎，肺炎可以在24-48小时内致死。如果你看到**嘴巴有泡沫**——这是急诊。

### 兽医会做什么？

1. 听诊——判断是否有肺炎（肺部有啰音）
2. 可能拍X光——确认肺炎范围和严重程度
3. 开抗生素（通常是注射型，如恩诺沙星）
4. 严重时——住院吸氧+输液

### 预防

- 维持正确温度梯度（热点28-30°C，冷区24-26°C）
- 通风良好（饲养箱不要完全密闭）
- 不要过度喷水（湿度过高）
- 新到家的守宫，前2周不要打扰——让它们适应，减少压力

## 四、代谢性骨病（Metabolic Bone Disease / MBD）

### 症状（按严重程度）

**早期**：
- 下巴变软（"橡胶下巴"）——用手指轻压下巴，正常应该是硬的
- 走路姿势异常——后腿看起来"无力"或"摇摆"
- 食欲下降

**中期**：
- 下巴明显变形（V形或W形）
- 四肢/脊柱弯曲
- 爬行困难，后腿拖地

**晚期**：
- 骨折（自发性——轻微碰撞就断）
- 无法站立或爬行
- 拒食，极度消瘦

### 原因

**缺钙 + 缺维生素D3 = 钙吸收障碍 = 身体从骨骼调取钙质 = 骨骼软化**

具体原因：
- 没有UVB灯（或UVB灯老化/衰减）
- 食物中不加钙粉
- 钙粉中不含D3，且宠物没有UVB光源
- 食物中磷含量过高（磷会阻碍钙吸收）

### 家庭处理 vs 看兽医

**MBD是医疗急症。任何阶段都应该看兽医。**

但如果你发现了早期症状（下巴变软、走路异常），你可以做的：

1. **立即增加钙补充**——食物上撒钙粉（含D3），每次喂食都撒
2. **立即检查UVB灯**——可能坏了或衰减了，换新的
3. **提高温度**——29-30°C促进新陈代谢和钙代谢
4. **立即看兽医**——早期MBD可以逆转，中晚期不可逆

**警告**：MBD导致的骨骼变形是**永久性**的。早期发现+治疗可以阻止恶化，但已经变形的骨骼无法恢复。这就是为什么预防比治疗重要一万倍。

### 兽医会做什么？

1. 触诊——检查下巴、四肢、脊柱的柔软度
2. X光——确认骨骼软化和骨折情况
3. 血钙检测——确认血钙水平
4. 治疗：
   - 早期：口服或注射钙剂 + 纠正饲养
   - 中期：注射钙剂 + 可能住院
   - 晚期：手术固定骨折 + 长期康复

### 预防（这是唯一正确的态度）

- **UVB灯不能省**——守宫需要UVB来合成维生素D3
- **钙粉每次喂食都要撒**——薄薄一层，覆盖昆虫表面
- **钙磷比要正确**——食物（昆虫）的钙磷比应该接近1:1到1:2
- **不要用松木屑**——松木屑中的油脂会干扰钙代谢

## 五、寄生虫（Internal Parasites）

### 症状

- 食欲正常或增加，但体重下降（"能吃不长肉"）
- 便便中有虫体（白色线状，0.5-2cm长）
- 便便中有黏液或带血
- 活动量下降

### 原因

- **野生捕获个体（WC）**——寄生虫感染率接近100%
- 食物污染（喂了带寄生虫卵的昆虫）
- 环境交叉污染（从宠物市场/爬宠店带回了卵）

### 家庭处理 vs 看兽医

**寄生虫必须看兽医。**

不要自己网购驱虫药给守宫吃——剂量计算错误可能致死。异宠兽医会根据粪检结果选择对的药和对的剂量。

### 兽医会做什么？

1. 粪检——显微镜下找虫卵，确定寄生虫种类
2. 开驱虫药——根据寄生虫种类选择（芬苯达唑/Flubendazole/Metronidazole等）
3. 复查——治疗结束后2-3周复查粪检，确认寄生虫已清除

### 预防

- **只买CB（人工繁育）个体**——WC个体寄生虫感染率极高
- 新到家的守宫，隔离观察30天 + 做粪检
- 食物来源可靠（不要喂野外抓的昆虫）
- 定期（每6-12个月）做粪检——即使没有症状

## 六、口腔炎（Stomatitis / "Mouth Rot"）

### 症状

- 嘴巴里面有白色、黄色或棕色的坏死组织
- 嘴巴无法完全闭合（因为肿胀或坏死组织）
- 口水过多（正常守宫几乎没有口水）
- 拒食（嘴巴疼，无法咬食物）
- 嘴巴周围有干涸的分泌物

### 原因

- **外伤**——食物（如乳鼠）咬了口腔，或垫材（沙子）刮伤
- 免疫力下降（温度不对、压力、其他疾病）
- 细菌感染（继发性）

### 家庭处理 vs 看兽医

**口腔炎必须看兽医。**

这不是你在家能处理的问题。口腔炎如果不治疗，感染会扩散到下颌骨，导致骨髓炎，最终可能需要截除部分下颌。

### 兽医会做什么？

1. 麻醉下检查口腔——评估感染范围和深度
2. 取样做细菌培养+药敏试验——确定用什么抗生素最有效
3. 清创——清除坏死组织
4. 开抗生素（口服或注射）+ 口腔冲洗液（如洗必泰/Chlorhexidine）
5. 严重时——住院静脉抗生素

### 预防

- 食物大小合适（不要喂太大的食物，避免嘴巴外伤）
- 喂乳鼠时用长镊子——不要让活鼠和守宫共处
- 垫材不用沙子（沙子会刮伤口腔）
- 维持正确温度——守宫免疫系统需要28-30°C

## 总结

守宫的疾病，99%是**饲养不当**引起的。你作为饲养者，最重要的不是"学会了怎么治疗"，而是"学会了怎么不让它生病"。

**核心预防原则**：
1. 温度对（热点28-30°C）
2. 湿度对（50-60%）
3. 食物干净（来源可靠 + 钙粉）
4. UVB灯正常工作（每6个月换）
5. 定期观察（每周检查一次守宫的身体和便便）

如果你发现异常，不要等——**早一天看兽医，可能救一条命。**
