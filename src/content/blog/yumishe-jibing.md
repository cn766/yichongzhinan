玉米蛇总体来说是皮实的宠物，但在不当的饲养环境下，它们一样会生病。好消息是——**90%的疾病与饲养环境直接相关**，这意味着你在家就能预防和解决大部分问题。

本文覆盖8种玉米蛇最常见疾病，教你识别、处理和预防。

---

## ⚠️ 重要：何时必须看兽医？

在你开始自我诊断之前，先明确什么情况必须看兽医：

| 紧急程度 | 症状 | 行动 |
|:---:|------|------|
| 🔴 立即就诊 | 持续张嘴呼吸、口腔内有奶酪状物、身体有开放性伤口或脓肿、严重脱水（皮肤捏起不回弹） | 不要等，马上找异宠兽医 |
| 🟡 尽快就诊 | 持续拒食+体重下降超过10%、眼睛或口腔有分泌物、粪便持续异常超过2周 | 预约最近的爬宠兽医 |
| 🟢 可先在家处理 | 脱皮不全、轻微外伤（无感染）、偶尔一次呕吐、轻度拒食 | 先排查饲养环境+温度 |

## 疾病1：呼吸道感染（Respiratory Infection）

### 症状识别
- 🚩 **张嘴呼吸**（最有标志性的信号）
- 🚩 呼吸时有"嘶嘶"声或"咔嗒"声（不是蛇的正常嘶嘶声！）
- 🚩 鼻孔有气泡或粘液
- 🚩 头部抬高保持（试图让呼吸更顺畅）
- 🚩 食欲减退或停止进食

### 原因
| 主要原因 | 说明 |
|----------|------|
| **温度过低** | **最常见原因**——热点<26°C持续数天，免疫系统被抑制 |
| **湿度过高** | 湿度持续>80%，细菌容易繁殖 |
| **通风差** | 氨气/霉菌在封闭环境积累 |
| **受凉** | 冷风直吹、昼夜温差过大 |

### 家庭处理（轻度）

**如果症状较轻**（偶尔张嘴呼吸，但仍然进食），你可以：

1. **立即提高温度**：热点从常规的30°C提高到**33-34°C**（这是关键！更高的温度加速免疫反应）
2. **降低湿度**：停止喷水，确保垫材干燥
3. **加强通风**：确保饲养箱有足够的通风孔
4. **48小时观察**：如果症状没有改善或恶化 → 立即就医

**❌ 不要做的事**：
- 不要用任何人的感冒药/抗生素（蛇的剂量跟人完全不同！）
- 不要用任何偏方（大蒜水、醋水等）
- 不要继续喂食（消化会进一步压迫呼吸）

### 兽医治疗（中重度）

兽医通常会：
- 进行细菌培养+药敏测试
- 处方爬宠专用抗生素（通常需要注射）
- 可能需要进行雾化治疗

### 预防

```
保持热点 29-32°C 范围内稳定运行
→ 保持湿度 40-60%
→ 确保适度通风
→ 每天检查温度计
→ 呼吸道感染几乎不会发生
```

## 疾病2：口腔炎/口腐病（Mouth Rot / Infectious Stomatitis）

### 症状识别
- 🚩 口腔内有**白色/黄色/奶酪状物质**（标志性症状）
- 🚩 牙龈红肿、出血
- 🚩 嘴角有泡沫或粘液
- 🚩 嘴巴无法完全闭合
- 🚩 食欲减退

### 原因
- 口腔外伤（被活鼠咬伤、撞到硬物）
- 温度过低+免疫力下降
- 维生素缺乏
- 应激

### 处理方式

**口腔炎需要看兽医，不能自行处理。** 兽医需要：
1. 清理坏死组织
2. 局部用药（碘伏/氯己定冲洗）
3. 全身抗生素治疗

### 预防
- 用冻鼠不用活鼠
- 饲养箱内不要放尖锐物品
- 保持温度稳定

## 疾病3：鳞片腐烂/水疱病（Scale Rot / Blister Disease）

### 症状识别
- 🚩 腹部鳞片发红、肿胀
- 🚩 鳞片下有液体（水泡）
- 🚩 严重的鳞片会脱落露出下面的皮肤
- 🚩 通常从腹部开始，因为腹部接触潮湿垫材

### 原因

**唯一原因**：蛇长期待在**过湿的垫材**上。垫材被尿液/粪便浸泡后没有及时更换，细菌滋生。

### 处理方案

**轻度（几个鳞片发红，无水泡）**：
1. 立即更换所有垫材为干燥的新垫材
2. 降低湿度到40-50%
3. 用水盆替代喷雾加湿
4. 每天检查，症状应在1周内改善

**中重度（有水泡/鳞片脱落/有异味）**：
→ **立即就医**。需要抗生素治疗，可能还需要局部清创。

### 预防
```
保持垫材干燥 + 及时清理粪便 = 不会得鳞片腐烂
```

## 疾病4：体内寄生虫

### 症状识别
- 🚩 粪便中有**可见的蠕虫**（蛔虫/线虫等）
- 🚩 尽管正常进食但体重不增或减轻
- 🚩 腹泻、粪便恶臭
- 🚩 腹部膨胀

### 来源
- 吃了带寄生虫的活鼠
- 接触了被污染的垫材/水源
- 从繁殖者那里就携带的（最常见）

### 处理方式

**需要兽医用显微镜做粪便检查确认寄生虫种类**，然后开对应的驱虫药。不同寄生虫用不同的药，不能乱用。

### 是否需要"定期驱虫"？

**不需要。** 只有粪便检查确认有寄生虫时才驱虫。盲目驱虫只会增加蛇的肝脏负担。

## 疾病5：便秘/排便困难

### 症状识别
- 蛇持续拱背、轻微颤抖
- 泄殖腔处有硬块
- 超过2-3周没有排便（成体正常可能更久）
- 食欲减退

### 原因
- 饮水不足
- 温度偏低（减缓肠道蠕动）
- 食物尺寸过大
- 垫材被误食（尤其是细小的沙粒垫材）

### 处理方案

1. **温水浴**：水温30-32°C，浸泡15-20分钟（刺激肠道蠕动）
2. **确保水盆有足够清水**：有些蛇喜欢在水里排便
3. **检查温度**：确认热点不低于29°C
4. **如果48小时未改善**：找兽医

**❌ 绝对不要自己用工具帮蛇"通便"**——泄殖腔非常脆弱，容易造成严重损伤。

## 疾病6：脱皮不全（Dysecdysis）

这是新手最容易遇到也最不用慌的问题。

### 症状识别
- 原本应该完整脱下的蛇皮变成了碎片
- 身体上残留斑块状旧皮（尤其是眼睛部位）
- 尾巴末端存留旧皮（可能导致缺血坏死！）

### 原因

| 主要原因 | 占比 |
|----------|:---:|
| **湿度太低** | 90% |
| 营养不良 | 5% |
| 体表有伤疤 | 3% |
| 体外寄生虫 | 2% |

最常见就是 **"湿度不够"**。

### 处理方案

**第1步：检查是否全部脱完**
- 检查**眼睛**（眼罩是否完整脱掉）
- 检查**尾巴尖**（最容易残留，最危险）

**第2步：增加湿度**
- 提供一个**湿润隐藏洞**：在一个遮蔽物里放湿润的水苔/纸巾
- 可以用喷雾瓶把遮蔽物内部喷湿
- 在饲养箱里放一个"湿度箱子"

**第3步：帮助去除残留皮**
- 温水浴（30-32°C）15-20分钟
- 用湿毛巾轻轻擦拭残留的皮
- **⚠️ 绝对不要用手硬撕眼睛上的旧皮！** 会损伤角膜！
- 眼睛位置的眼罩残留：泡温水15分钟+用湿棉签轻轻滚动（不是擦拭）→ 如果不行就找兽医

**尾巴尖旧皮残留的紧迫性**：

> 尾巴尖的旧皮如果长期不处理，会像止血带一样勒住尾巴，最终导致**缺血坏死**，尾巴尖端脱落。所以每次蜕皮后检查尾巴尖是必修课。

### 预防
- 蜕皮期（眼睛变蓝→蜕皮前）保持湿度在60-70%
- 提供湿润遮蔽处
- 蜕皮完成后立即检查完整性

## 疾病7：呕吐/反流（Regurgitation）

### ⚠️ 这是严重问题，需要比拒食更警惕！

### 呕吐 vs 正常排便
| | 呕吐物 | 粪便 |
|------|------|------|
| **外观** | 半消化/完整的老鼠形状 | 黑/棕色糊状或固体 |
| **气味** | 明显腐烂味 | 普通粪便味 |
| **位置** | 通常在热区附近（蛇试图消化但失败了） | 随机位置 |
| **严重性** | 🔴需要警惕 | 🟢正常 |

### 为什么呕吐很严重？

蛇呕吐时会同时吐出**胃酸和消化酶**，这些东西会灼伤食道。而且一次呕吐后，蛇的消化系统非常脆弱，如果再次进食可能导致更严重的呕吐。

### 呕吐后的处理流程

```
Day 1（发现呕吐）：
  - 清理呕吐物
  - 不要喂任何东西
  - 确保热点32-34°C（帮助恢复）

Day 2-7：
  - 静静观察，完全不打扰
  - 保持温度恒定
  
Day 8-10：
  - 确认温度稳定、环境安静
  - 尝试喂一只比平时小一号的食物（如平时喂成鼠，现在喂跳鼠）
  
Day 11-12：
  - 如果不吐，进入恢复期
  - 如果又吐了 → 立即就医
```

**⚠️ 最少等7天！！！** 这是最重要的一条。不要心软提前喂——你的"心疼"会害死蛇。

### 原因
- 温度不够（最常见）
- 食物太大
- 喂食后太快上手
- 压力/紧张
- 体内有潜在疾病

## 疾病8：肥胖症

### 你可能正在把蛇喂成"胖子"

**如何判断你的蛇是否肥胖？**

| 体型 | 脊椎 | 身体横截面 | 尾巴到身体的过渡 |
|------|------|-----------|-----------------|
| 偏瘦 | 脊椎明显突出 | 三角形 | 突然变细 |
| 健康 | 可触摸到但不突出 | 圆形 | 平滑过渡 |
| 肥胖 | 摸不到脊椎 | 椭圆或扁平 | "臀部"肥肉堆积 |

另外：如果你的成体蛇的身体两侧有明显"肉褶"（尤其是弯曲时） → 肥胖警告。

### 处理

- 成体蛇从每14天改到每21天喂一次
- 减小食物尺寸一档
- 增加饲养箱空间让蛇多活动

## 疾病总结速查表

| 疾病 | 关键症状 | 首要处理 | 是否必须就医 |
|------|----------|----------|:---:|
| 呼吸道感染 | 张嘴呼吸、有呼吸声 | 提高温度至33°C | 中重度必须 |
| 口腔炎 | 口腔内奶酪状物 | 立即就医 | ✅ 必须 |
| 鳞片腐烂 | 腹部鳞片发红/水泡 | 换干燥垫材 | 中重度必须 |
| 体内寄生虫 | 粪便可见虫体 | 粪便检查+驱虫药 | ✅ 必须 |
| 便秘 | 泄殖腔硬块 | 温水浴 | 持续不排便才去 |
| 脱皮不全 | 碎片状旧皮 | 增加湿度+湿润遮蔽处 | 极少需要 |
| 呕吐 | 半消化食物 | **禁食至少7天** | 反复呕吐必须 |
| 肥胖 | 摸不到脊椎 | 延长喂食间隔 | 通常不需要 |

## 预防 > 治疗：建立日常健康检查习惯

每周花5分钟检查这些项目：

- [ ] 温度计读数（热点29-32°C，冷区24-26°C）
- [ ] 湿度计读数（40-60%）
- [ ] 蛇的外观（眼睛明亮、鳞片光滑、无伤口）
- [ ] 粪便状况（成形、无异色、无虫体）
- [ ] 最近一次蜕皮是否完整
- [ ] 饮水是否充足（水盆有清水）
- [ ] 行为是否正常（不长时间在冷区蜷缩）

每天做这些检查 = 99%的问题在萌芽阶段就被发现了。
