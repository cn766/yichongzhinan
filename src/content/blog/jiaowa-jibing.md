## 疾病1：红腿病（Red Leg Syndrome）—— 角蛙的"绝症"

### ⚠️ 这是角蛙的头号杀手

**红腿病是两栖动物的细菌性败血症**——全身细菌感染。一旦发展到肉眼可见的红腿，死亡率极高。

### 病原
- 细菌（最常见：气单胞菌 *Aeromonas*）
- 通过不洁的环境/水源侵入

### 症状
- 🚩 **腹部和/或大腿内侧出现大面积红色**（标志性症状）
- 🚩 精神萎靡、几乎没有反应
- 🚩 身体瘫软、不挣扎
- 🚩 皮肤表面可能有出血点

### 处理
→ **立即就医。没有任何家庭处理方法。** 需要抗生素注射或药浴（兽医处方）。红腿病从出现症状到死亡可能只有24-48小时。

### 预防
- **保持环境和水源清洁**（每天换水！）
- 垫材每周更换
- 不要让蛙长时间泡在自己的排泄物里

## 疾病2：肠道堵塞（Impaction）

### 为什么角蛙特别容易肠道堵塞？

角蛙是"坐等型"猎手——它们蹲在垫材上，任何路过嘴边的东西都可能被一口吞下，**包括垫材本身**。

### 原因

| 原因 | 说明 |
|------|------|
| **松散的沙质垫材** | 头号元凶——捕食时吞入沙子/细砾 |
| 吃了面包虫/大麦虫（壳不消化） | 几丁质在肠道积累 |
| 食物过大 | 堵塞肠道 |

### 症状
- 🚩 多日不排便
- 🚩 腹部摸到硬块
- 🚩 食欲废绝
- 🚩 身体膨胀或不对称

### 家庭处理

**轻度**（还能排便、只是排便减少）：
1. 温水浴（26-28°C，15分钟）
2. 轻柔按摩腹部
3. 如果能买到角蛙专用，可以尝试给一小滴矿物油口服（润滑肠道）
4. 停止喂食直到正常排便

**中度以上**（多日不排便+腹部硬块）：
→ **立即就医。** 兽医可能需要灌肠。

### 预防
```
永远不用沙质/细碎垫材
  → 捕食时用镊子精准投喂（不让食物掉到垫材上）
  → 或把角蛙移到空盒里喂食
```

## 疾病3：气肿症（Edema / Bloat）

### 症状
- 🚩 身体像气球一样膨胀（不只是"胖"，是**充气的**）
- 🚩 皮肤紧绷、透明感
- 🚩 严重时可以看到皮下液体或气体

### 原因
- 肾功能衰竭
- 细菌感染
- 寄生虫
- 淋巴系统异常

### 处理
→ **立即就医。** 气肿症需要兽医诊断根本原因。皮下液体多时兽医可以用无菌针头穿刺排出——但**千万不要自己用针扎**，会引入感染。

## 疾病4：真菌感染

### 症状
- 皮肤上出现**白色/灰色棉絮状斑块**
- 常见于背部或四肢
- 斑块下方的皮肤可能红肿

### 处理

**轻度（小块白斑）**：
- 环境排查：垫材是否过湿？通风够不够？
- 降低湿度到60-70%
- 可以用兽医处方的抗真菌药浴

**中度以上（大面积或扩散）**：
→ **就医。**

### 预防
- 保持环境不过湿
- 垫材定期更换
- 提供适度通风

## 疾病5：代谢性骨病（MBD）

角蛙也会得MBD，虽然比鬃狮蜥少见。

### 症状
- 🚩 **下颚下垂、嘴巴合不拢**（最明显的信号）
- 🚩 四肢无力、跳跃困难
- 🚩 脊椎弯曲

### 原因
- 长期缺钙
- 缺维生素D3
- 钙磷比失衡的饮食

### 处理

**早期**（下颚还能闭合，只有轻微无力）：
- 立即加强钙粉补充（每顿都蘸）
- 确保喂食场所在有UVB的环境（角蛙不一定需要UVB灯，但有比没有好）

**中重度**（下颚下垂明显、无法正常闭合）：
→ **就医。** 需要兽医进行补钙治疗。**下颚变形通常不可逆。**

## 疾病6：脱水

### 症状
- 皮肤干燥、失去光泽
- 身体变薄变平（不是正常的圆润）
- 眼睛凹陷
- 触摸它时反应迟钝

### 处理

1. 立即放入26-28°C的温水中浸泡15-20分钟
2. 检查环境湿度（应该是70-80%！）
3. 确保水盆有清水
4. 增加喷水频率

**严重脱水**（皮肤严重干燥+完全不活跃）→ 就医，可能需要皮下补液。

## 日常健康检查（30秒）

```
今天：
  - 角蛙的姿势正常吗（不是歪斜或瘫软）？
  - 腹部和腿部没有发红？
  - 皮肤湿润有光泽？
  - 水盆的水是干净的吗？
  - 垫材湿度对吗（不积水、不过干）？
```

30秒的检查，能避免90%的疾病发展到不可控阶段。角蛙不会发出声音求救——它们的"沉默"需要你来打破。
