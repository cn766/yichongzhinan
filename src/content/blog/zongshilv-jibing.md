鬃狮蜥的疾病绝大多数是**饲养环境不当**导致的——温度不够、UVB不足、饮食不当。这意味着大部分疾病是可以预防的。但一旦发生了，你需要知道怎么识别和处理。

---

## ⚠️ 就医判断标准

| 紧急程度 | 症状 | 行动 |
|:---:|------|------|
| 🔴 立即就医 | 持续张嘴呼吸、四肢颤抖抽搐、下颌明显歪斜变形、有脓包/肿胀、瘫痪 | 不要等 |
| 🟡 尽快就医 | 持续拒食+体重明显下降、腹泻超过1周、行为异常（长期不动） | 预约异宠兽医 |
| 🟢 可家庭观察 | 偶发性软便、轻微脱皮不完整、轻度拒食1-2天 | 先排查环境 |

## 疾病1：代谢性骨病（MBD）—— 头号杀手

### 为什么MBD是鬃狮蜥的第一大病？

MBD不是一种独立的"病"，而是**因为缺钙引起的一系列骨骼代谢异常的总称**。它是鬃狮蜥中最常见也最可预防的疾病。

### 病原链

```
UVB不足（或完全无UVB灯）
  → 皮肤无法合成维生素D3
  → 肠道无法吸收钙质
  → 身体开始从骨骼中"抽走"钙来维持血钙
  → 骨骼软化、变形、易骨折
```

### 症状分级

| 阶段 | 症状 |
|:---:|------|
| **早期** | 走路摇晃、后肢轻微颤抖、食欲下降 |
| **中期** | 下颌变软（像橡胶）、四肢弯曲/变形、行动困难、经常趴着 |
| **晚期** | 下颌严重歪斜（永久性变形）、自发性骨折、抽搐、无法站立 |
| **终末期** | 全身瘫痪、无法进食、死亡 |

### 处理方案

**轻中度MBD**（早期发现）：
1. **立即检查并加强UVB**——这是最优先的事
   - 检查灯管是否过期（超过12个月？）
   - 检查安装距离是否正确
   - 换高质量灯管（Arcadia T5 Dragon 12%）
2. **每日补充钙粉+含D3复合粉**
3. **确保晒点温度达标**（38-42°C，钙代谢需要高温）
4. **轻症1-2周后可见改善**

**中重度MBD**：
→ **必须就医。** 需要兽医进行血钙检测+处方高剂量钙剂注射。

### ⚠️ 关键警告

> **MBD造成的骨骼变形是不可逆的。** 补钙可以让蜥蜴停止继续恶化并恢复体力，但已经弯曲变形的腿和下颌不会恢复原状。

这就是为什么预防MBD比治疗MBD重要一万倍。

## 疾病2：呼吸道感染

### 症状
- 张嘴呼吸（最关键的信号）
- 鼻孔有气泡或粘液
- 呼吸声沉重
- 泡沫从嘴/鼻流出
- 活动减少

### 原因

| 原因 | 占比 |
|------|:---:|
| 温度过低 | 70% |
| 湿度过高 | 20% |
| 通风不良 | 10% |

### 处理

**轻度**：
- 立即将温度提高到晒点40-42°C、冷区28-30°C
- 降低湿度（<30%）
- 24-48小时观察

**中重度**：
→ **就医**。需要处方抗生素。

## 疾病3：体内寄生虫

### 症状
- 粪便恶臭
- 腹泻
- 食欲正常但越来越瘦
- 腹部胀大
- 嗜睡、精神差

### 诊断

需要兽医做**粪便显微镜检查**确认寄生虫种类（球虫、蛲虫、鞭毛虫等）。

### 处理

不同类型寄生虫用不同的驱虫药。**不能自己买人用/猫狗用的驱虫药来喂。** 剂量完全不同。

## 疾病4：便秘与肠道堵塞（Impaction）

鬃狮蜥的肠道堵塞是比便秘更严重的状况——肠道被无法消化的物质堵死了。

### 原因

| 原因 | 说明 |
|------|------|
| **松散的沙质垫材被误食** | 头号元凶！进食时舔到沙子吞入 |
| 食物昆虫过大 | 肠道无法推进 |
| 温度过低 | 肠道蠕动减弱 |
| 吃了不能消化的东西 | 塑料、橡胶等 |

### 症状
- 多日不排便
- 后肢可能瘫痪（堵塞物压迫神经）
- 腹部有硬块可摸到
- 弓背、用力但排不出
- 食欲废绝

### 处理方案

**轻度（只便秘、无后肢症状）**：
1. 温水浴（32-35°C，30分钟）
2. 轻柔按摩腹部
3. 喂一小勺橄榄油或南瓜泥（润滑肠道）

**中重度（后肢拖行、腹部明显硬块）**：
→ **立即就医！** 这是危及生命的状况，可能需要灌肠甚至手术。

### 预防
```
❌ 不用松散沙质垫材
✅ 用厨房纸巾/石板/爬宠地毯/瓷砖
✅ 食物放在食盆里（不直接放垫材上）
✅ 保证晒点温度达标
```

## 疾病5：脓疡/脓肿

### 症状
- 体表出现局部肿胀，触感软或硬
- 可能破溃流出黄白色脓液
- 常见于头部、四肢关节

### 原因
- 外伤感染
- 被虫咬伤（活虫在饲养箱里咬蜥蜴）
- 垫材不洁

### 处理

→ **就医。** 爬行动物的脓液是**干酪状固体**（不同于哺乳动物的液态脓），需要兽医切开清理+抗生素治疗。自己挤可能会把感染推向深层。

---

## 疾病6：尾部坏死

### 症状
- 尾巴末端变黑、干瘪
- 逐渐向根部蔓延
- 黑色部分失去知觉

### 原因
- 蜕皮不完整→旧皮箍住尾巴→缺血
- 外伤→感染→组织坏死

### 处理
→ **就医。** 可能需要截除坏死部分，防止继续向上蔓延。

---

## 疾病7：脱水

### 症状
- 皮肤弹性变差（用手指轻轻捏起背部皮肤，回弹慢 = 脱水）
- 眼睛凹陷
- 尿液中的白色尿酸部分变硬、变色
- 精神萎靡

### 处理

1. 温水浴（32-35°C，15-20分钟）
2. 每天给含水分蔬菜（蒲公英叶、芥蓝）
3. 每天用喷雾瓶向鼻子喷水（它会舔水珠）
4. 严重的需要兽医皮下补液

## 疾病总结速查表

| 疾病 | 关键症状 | 首要处理 | 必须就医？ |
|------|----------|----------|:---:|
| MBD | 下颌变软、四肢颤抖 | 加强UVB+补钙 | 中重度必须 |
| 呼吸道感染 | 张嘴呼吸、有气泡 | 提高温度 | 中重度必须 |
| 体内寄生虫 | 粪便恶臭、消瘦 | 粪便检查 | ✅ 必须 |
| 肠道堵塞 | 不排便、后肢拖行 | 温水浴 | 中重度必须 |
| 脓疡 | 体表肿胀/脓包 | 就医切开 | ✅ 必须 |
| 尾部坏死 | 尾巴变黑变干 | 就医 | ✅ 必须 |
| 脱水 | 皮肤不回弹、眼凹陷 | 温水浴+喷水 | 严重时就医 |

## 日常健康检查checklist

每天3分钟：

- [ ] 晒点温度（38-42°C）和冷区温度（26-29°C）
- [ ] UVB灯亮着吗？安装日期在12个月内吗？
- [ ] 最后一次排便正常吗？
- [ ] 走路姿势正常吗？（不拖行、不颤抖、不歪斜）
- [ ] 精神状态活跃吗？（回应你的存在，而不是一直趴着）
- [ ] 眼睛明亮吗？呼吸安静吗？
- [ ] 食物和水还新鲜吗？

## 最重要的三句话

1. **MBD是鬃狮蜥的头号杀手，但它100%可以预防——只要UVB灯对、钙粉到位。**
2. **张嘴呼吸 = 危险信号，不要等。**
3. **尾巴尖的旧皮如果长期不处理，尾巴会坏死脱落。**

记住：养得好，鬃狮蜥可以活8-12年。养得不好，可能撑不过第一年。差别就在于你是否愿意每天花3分钟做健康检查。
