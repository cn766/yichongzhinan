角蛙是天生的"伏击猎手"——它们可以一动不动蹲几个小时，然后一口吞掉路过的一切能动的东西。但在人工饲养中，这个"一切能动的东西"需要你精心选择。

---

## 一、角蛙吃什么？

### 1.1 推荐主食

| 食物 | 推荐等级 | 营养 | 备注 |
|------|:---:|------|------|
| **蟋蟀** | ⭐⭐⭐⭐⭐ | 蛋白高、脂肪低 | 最方便的主食 |
| **杜比亚蟑螂** | ⭐⭐⭐⭐⭐ | 营养全面 | 比蟋蟀更好消化 |
| **蚯蚓** | ⭐⭐⭐⭐ | 天然高钙 | 不错的轮换选择 |
| **饲料鱼（小尺寸）** | ⭐⭐⭐ | 含Omega-3 | 不能天天喂 |
| **乳鼠** | ⭐⭐ | 高蛋白高脂肪 | **成体偶尔喂（每月1次），幼体不能喂** |

### 1.2 绝对不能喂的食物

| 禁止食物 | 原因 |
|----------|------|
| **面包虫/大麦虫** | 壳太硬→肠道堵塞风险极高 |
| **野外的虫子** | 农药、寄生虫 |
| **哺乳动物肉（猪肉/牛肉/鸡肉）** | 角蛙无法消化哺乳动物肉 |
| **金鱼**（作为主食） | 含硫胺素酶，长期导致维生素B1缺乏 |
| **死掉不动的食物** | 角蛙通常不吃死物 |

### 1.3 为什么面包虫对角蛙是"毒药"？

角蛙的消化系统比鬃狮蜥更"弱"。面包虫的几丁质外壳对角蛙的肠道来说极难消化，大量喂食 = 肠道堵塞。**这是角蛙最常见的死亡原因之一。**

## 二、喂食频率与量

| 年龄 | 喂食频率 | 每顿量 | 饵料大小 |
|------|:---:|------|------|
| **幼体（<4cm）** | 每天一次 | 2-4只小蟋蟀 | 不超过两眼间距 |
| **亚成体（4-8cm）** | 每1-2天一次 | 3-6只中蟋蟀或1-2只杜比亚 | 不超过嘴巴宽度的1/3 |
| **成体（>8cm）** | 每2-4天一次 | 4-8只大蟋蟀或2-3只大杜比亚 | 不超过嘴巴宽度的1/2 |

### 关键规则

1. **食物尺寸 < 蛙嘴宽度的1/2**。角蛙虽然嘴大能吞，但太大 → 消化不良
2. **吃饱就停。** 角蛙吃撑了会拒绝再吃。不要强迫喂食
3. **不要天天喂成体。** 肥胖是角蛙的慢性杀手

## 三、钙粉与维生素

| 喂食日 | 用什么粉 |
|:---:|------|
| 每周4天 | 纯钙粉（无D3） |
| 每周1-2天 | 含D3钙粉或爬宠复合维生素粉 |

**操作**：饵料放入杯中 → 撒粉 → 摇晃 → 立即喂食。

角蛙没有UVB灯也能通过食物获取D3（不像鬃狮蜥那么依赖UVB），但钙粉还是必须的——没有足够的钙，骨骼会出问题。

## 四、喂食实操

### 怎么喂？

角蛙是伏击型捕食者，它们识别食物的方式是**"看到动的 = 食物"**。

1. 用镊子夹住饵料
2. 放在角蛙面前3-5cm处
3. 轻轻晃动（模拟活物运动）
4. 角蛙会扑过来一口吞掉

**⚠️不要用手直接喂！** 角蛙咬合力惊人，而且嘴大——你的手指看起来可能像一只大蟋蟀。

### 角蛙拒绝吃怎么办？

| 原因 | 解决方案 |
|------|----------|
| 温度太低 | 角蛙消化温度要求25-28°C，低于22°C会拒食 |
| 湿度不对 | 保持垫材湿润但不积水 |
| 刚换环境 | 到新家前3天不喂，让它适应 |
| 蜕皮期 | 角蛙蜕皮时会吃掉自己的皮，这几天食欲下降正常 |
| 食物不动 | 用镊子让食物"跳舞" |
| 一直不吃 | 健康角蛙可以饿1-2周。如果超过2周+体重明显下降 → 就医 |

## 五、喂食翻车现场

### 翻车1："它爱吃面包虫，我天天喂"
结果：肠道堵塞 → 死亡。**面包虫最多偶尔当零食，不能当主食。**

### 翻车2："它看起来很饿，多喂点"
结果：肥胖 → 寿命缩短，还可能导致脂肪肝。

### 翻车3："野外抓的虫子更新鲜"
结果：农药中毒。永远只喂养殖的饵料。

### 翻车4："把虫子直接丢笼子里不看着"
结果：蟋蟀可能反过来啃咬角蛙的皮肤（尤其是幼体），造成外伤感染。
