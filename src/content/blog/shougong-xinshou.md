守宫是目前国内最受欢迎的异宠之一。它们体型小巧、不占空间、安静、不需要遛，养好了能陪你10-20年。

但对于新手来说，最大的坑不是"怎么养"，而是**买之前不知道要看什么**。很多人兴冲冲买回来，才发现饲养箱买小了、温控没装、垫材用错——然后守宫就开始拒食、蜕皮困难、甚至生病。

这份指南的目标，就是让你在带守宫回家之前，把所有准备工作一次性搞清楚。

---

## 选什么品种？三种新手友好守宫对比

新手最容易上手的三类守宫：豹纹守宫、睫角守宫、肥尾守宫。它们的饲养难度从低到高排列如下：

### 豹纹守宫（Leopard Gecko）⭐ 新手首选

| 项目 | 参数 |
|------|------|
| 成年体长 | 18-25cm |
| 寿命 | 10-20年 |
| 食性 | 纯肉食（昆虫） |
| 温度需求 | 热区32-34℃ / 冷区24-26℃ |
| 湿度需求 | 30-40%（极低） |
| 是否需要UVB | 非必需，但有UVB更好 |
| 性格 | 温顺，易上手，互动性好 |
| 市场价格 | ¥100-500（普通基因） |

**为什么最适合新手？** 豹纹守宫对湿度零要求——干燥环境基本不会养死。食性简单，只吃虫子，不需要切水果蔬菜。温度控制只要一个加热垫就能搞定。

### 睫角守宫（Crested Gecko）⭐⭐

| 项目 | 参数 |
|------|------|
| 成年体长 | 20-25cm（含尾） |
| 寿命 | 15-20年 |
| 食性 | 杂食（专用果泥+昆虫） |
| 温度需求 | 22-26℃（怕热不怕冷） |
| 湿度需求 | 60-80%（需每天喷水） |
| 是否需要UVB | 非必需 |
| 性格 | 温顺，跳跃力强，易断尾 |
| 市场价格 | ¥300-1500 |

**优点**：不用活虫也没问题，有专用果泥粉，冲水就行。颜值高，花纹变化丰富。**缺点**：需要每天喷水维持湿度，夏天超过28℃需要降温措施。睫角守宫一旦断尾不会再生。

### 肥尾守宫（African Fat-Tailed Gecko）⭐⭐

| 项目 | 参数 |
|------|------|
| 成年体长 | 18-23cm |
| 寿命 | 15-20年 |
| 食性 | 纯肉食（昆虫） |
| 温度需求 | 热区31-33℃ / 冷区23-25℃ |
| 湿度需求 | 50-60% |
| 是否需要UVB | 非必需 |
| 性格 | 极其温顺，不太爱动 |
| 市场价格 | ¥400-2000 |

**肥尾守宫可以理解为"需要一点湿度的豹纹守宫"**。比豹纹胖一点、慢一点、更不爱动，是真正的懒人宠物。颜值上通常比豹纹更"憨"，花纹偏斑点和条纹。

> **新手建议**：从豹纹守宫开始。它是最容错的品种——忘喷水没事、温度波动一点没事、偶尔出差两天也没事。

## 饲养环境搭建

这是新手翻车最多的环节。分开讲三种守宫的共同点和差异点。

### 饲养箱

| 品种 | 最小底面积 | 推荐尺寸 | 箱体类型 |
|------|:---:|------|------|
| 豹纹守宫（单只成体） | 50×30cm | 60×40×30cm | 爬虫饲养箱（玻璃/亚克力） |
| 睫角守宫（单只成体） | 40×30cm | 45×45×60cm | **竖高箱**（它们需要攀爬高度） |
| 肥尾守宫（单只成体） | 50×30cm | 60×40×30cm | 爬虫饲养箱（玻璃/亚克力） |

> 核心原则：守宫是独居动物，**永远不要合养**。两只同性会打架，异性会持续繁殖导致雌性营养耗竭。
> 幼体可以先用 30×20cm 的小饲养盒，大了再升级。

### 加热设备

**豹纹守宫 & 肥尾守宫**：使用**底部加热垫**，贴在饲养箱底部外侧，面积占箱底的 **1/3**。必须在加热垫上装**温控器**——探头贴在热区底部，设定 32-34℃。

**睫角守宫**：通常情况下不需要额外加热。室温 22-26℃ 是它们的舒适区间。如果冬天室温低于 18℃，使用低功率陶瓷加热灯（配温控器）。

> 温控器不是可选设备，是**必须**。没有温控的加热垫可能烧到 40℃+，直接烫伤守宫腹部。

### 垫材

| 品种 | 推荐垫材 | 绝对禁止 |
|------|------|------|
| 豹纹守宫 | 厨房纸巾（幼体期）、爬虫沙/钙沙（成体） | 松木屑、雪松屑（含酚类毒素）、猫砂 |
| 睫角守宫 | 椰土、树皮碎片、苔藓 | 爬虫沙（误食风险） |
| 肥尾守宫 | 厨房纸巾（幼体）、椰土+沙混合（成体） | 纯钙沙（误食风险高） |

新手建议：**先用厨房纸巾（无香型）**，养三个月后再考虑换天然垫材。厨房纸巾便宜、好清理、无误食风险，是最安全的新手垫材。

### 躲避穴 & 湿度盒

每个饲养箱内至少放 **2个躲避穴**：一个在热区，一个在冷区。躲避穴用小号的陶盆、椰子壳、或者专门的爬宠躲避洞都行。

豹纹守宫和肥尾守宫还需要一个**湿度盒**：小盒子里放湿润的苔藓或蛭石，帮助蜕皮。湿度盒放在热区和冷区中间位置。

睫角守宫的湿度靠**每天喷水**维持——早晚各喷一次，使箱内湿度达到 60-80%，然后让它自然回落到 50% 左右。

### 水盆 & 食盆

- **水盆**：浅而稳，不易打翻。守宫不大会游泳，水太深有溺水风险。每天换新鲜水。
- **食盆**：如果喂钙粉/维生素粉，需要一个小食盆。睫角守宫需要果泥盆，最好放在高处（它们是树栖的）。

## 喂食方案

### 豹纹守宫 & 肥尾守宫

**主食**：蟋蟀、面包虫、杜比亚蟑螂

| 年龄段 | 喂食频率 | 每顿量 | 食物大小 |
|--------|:---:|------|------|
| 幼体（0-3月） | 每天1次 | 5-10只小虫 | 不超过守宫两眼间距 |
| 亚成体（3-8月） | 隔天1次 | 8-12只中虫 | 不超过守宫两眼间距 |
| 成体（8月+） | 每2-3天1次 | 6-10只成体虫 | 中等蟋蟀/杜比亚 |

**补钙方案**（比喂食本身更重要）：

- 每顿喂食前，把虫子放进钙粉袋里"裹粉"，摇一摇再喂
- 钙粉选择：**含D3的爬虫专用钙粉**（如 Repashy Calcium Plus 或 Zoomed Repti Calcium with D3）
- 如果饲养箱有 UVB 灯，可以用不含 D3 的钙粉
- 每周额外补充 1 次复合维生素粉（含维生素 A）

> 缺钙是豹纹守宫最常见的新手杀手。典型症状：走路摇摇晃晃、下巴变软、四肢颤抖。缺钙到后期不可逆，**预防远好于治疗**。

### 睫角守宫

睫角守宫可以用**完全不用活虫**的方式饲养，因为有专用果泥粉。

**主食**：睫角守宫专用果泥（Pangea 或 Repashy 品牌）
- 粉和水按 2:1 调成糊状
- 每 1-2 天换一次
- 每次放一小勺就够了

**加餐**（可选）：每周 1-2 次活虫（蟋蟀），裹钙粉后喂
- 活虫能增加蛋白质摄入，但不是必须

### 饲料虫的饲养

买来的虫子自己也要养——养好了才有营养喂守宫：

- **面包虫**：放麦麸+胡萝卜片，常温即可
- **蟋蟀**：放蛋托纸+胡萝卜/土豆片，喂之前24小时给蟋蟀"装填营养"（gut-loading），喂高蛋白饲料
- **杜比亚蟑螂**：放蛋托纸+狗粮碎+蔬果，繁殖慢但营养最好

> 一句话记住：你喂虫子的东西，就是你喂守宫的东西。虫子的营养=守宫的营养。

## 日常护理

### 清理频率

| 任务 | 频率 |
|------|------|
| 清理粪便 | 每天看到就清 |
| 换水 | 每天 |
| 换果泥（睫角） | 每1-2天 |
| 全箱大扫除 | 每月1次（换垫材+消毒） |

### 上手与互动

守宫不是猫狗，不要期待它们"喜欢被摸"。但豹纹守宫和肥尾守宫通常很能接受温和的上手。

**上手三步法：**

1. **第1周**：不要碰它。让它适应新环境。每天在箱外看看它，让它熟悉你的存在
2. **第2周**：把手放在箱内不动，让它自己爬上来探索。如果它紧张（尾巴快速摆动），收回手
3. **第3周+**：每次上手 5-10 分钟，不要超过 15 分钟。在沙发或床上上手，底下有软垫，防止摔伤

> 守宫上手最重要的规则：**永远不要抓尾巴**。豹纹守宫和肥尾守宫可以断尾求生，断尾虽然能再生一部分，但再生的尾巴短、丑、且消耗大量能量。睫角守宫的尾巴断后**不会再生**。

### 蜕皮

健康的守宫每 2-4 周蜕一次皮，幼体蜕得更频繁。正常情况下它们会自己把皮吃掉，你甚至不一定能看到蜕皮过程。

**蜕皮出问题的信号**：脚趾尖、尾巴尖有残留的老皮，像戴了个小"手套"。这说明湿度不够。解决方法：

- 放在湿度盒里泡 15-20 分钟（温水，刚没过脚面）
- 用湿润的棉签轻轻擦掉残留老皮
- 检查垫材和喷水频率，提高湿度

## 常见疾病

| 疾病 | 症状 | 原因 | 处理 |
|------|------|------|------|
| 缺钙/代谢性骨病 | 走路不稳、下巴软、肢体颤抖 | 缺钙+缺D3 | 立即补钙+D3，加UVB灯，严重就医 |
| 肠梗阻 | 拒食、腹部肿胀、不排便 | 误食垫材（沙/大块树皮） | 温水泡澡，滴橄榄油促排，严重需手术 |
| 呼吸道感染 | 张嘴呼吸、嘴边有泡、精神萎靡 | 温度太低+湿度过高 | 提高温度至35℃，降低湿度，就医 |
| 腐尾病 | 尾巴发黑、干枯、有异味 | 外伤感染或蜕皮残留 | 必须就医，可能需要截尾 |
| 卡皮 | 脚趾/尾尖有残留的蜕皮 | 湿度太低 | 温水泡+棉签清除，改善湿度 |

**需要立刻就医**的情况：守宫连续拒食 1 周以上、行动明显异常、口腔有分泌物、四肢明显变形。

## 新手常见误区

1. **"我先买个便宜的试试"** —— 守宫的购买成本只占总成本的 20-30%。设备不到位导致的看病费用（一次异宠诊所挂号费 ¥200-500）远超买一只好守宫的钱。

2. **"用旧鱼缸养也行"** —— 鱼缸通风差，容易发霉。爬虫饲养箱有侧通风网，一定要用专门的。

3. **"淘宝买的"新手套装"才 ¥100，省事"** —— 这种套装通常缺温控器，或者温控器质量极差。温控器坏了，加热垫不控温，直接烧到 40℃+，轻则烫伤，重则热死。

4. **"喂面包虫就够了"** —— 面包虫壳硬、营养单一、钙磷比差。要多种虫子轮换（蟋蟀+杜比亚+面包虫），配合裹钙粉。

5. **"守宫不需要看病"** —— 异宠也有兽医。大城市的异宠诊所可以治疗常见疾病。延误治疗比你想象的花费高得多。

## 下一步读什么？

这篇指南让你知道了"需要准备什么"。接下来你可以：

- 确定你要养哪个品种（建议从豹纹守宫开始）
- 采购设备，搭建好饲养箱后让它运行 48 小时，确认温湿度稳定
- 研究当地靠谱的守宫卖家或领养渠道
- 提前找到最近的异宠诊所地址和电话

养守宫是一段 10-20 年的陪伴。花几天时间做好功课，好过未来几年的焦虑和后悔。

> 最后一句话：**守宫不会爱你，但这不意味着你不该好好对它们**。它们用自己的方式活着，安静、优雅、不需要讨好任何人。这正是很多人爱上它们的原因。
