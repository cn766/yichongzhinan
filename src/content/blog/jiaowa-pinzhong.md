角蛙（Pacman Frog）可能是异宠界里最"反差萌"的存在——圆滚滚像一个有嘴的包子，平时一动不动，但捕食的瞬间快如闪电。在这张"大嘴萌"的外表下，其实是一个完整的品系世界。

---

## 一、角蛙家族三大原生种

角蛙属（*Ceratophrys*）有8个物种，但宠物市场上你主要会接触到这3种：

| 物种 | 英文名 | 体型 | 特征 |
|------|------|:---:|------|
| **南美角蛙** | *C. cranwelli* | 中等（10-15cm） | 最常见的宠物角蛙，大多数变异色型来自本种 |
| **钟角蛙** | *C. ornata* | 较大（12-17cm） | 颜色更鲜艳，眉角更高耸 |
| **霸王角蛙** | *C. cornuta* | 最大（15-20cm） | 头顶的"角"最长最高，体型最霸气 |

## 二、6大热门品种详解

### 1. 普通绿角蛙（Normal Green）
- **外观**：绿色基底 + 深绿色/棕色花纹
- **物种**：南美角蛙
- **特点**：最皮实，价格最低
- **价格**：¥50-150
- **推荐**：⭐⭐⭐⭐⭐ 新手必选

### 2. 白化角蛙（Albino）
- **外观**：黄色/橙色基底 + 橙色花纹，红色眼睛
- **物种**：南美角蛙（最常见）、钟角蛙
- **特点**：鲜艳夺目，是角蛙中最受欢迎的变异色
- **价格**：¥100-300
- **推荐**：⭐⭐⭐⭐⭐

### 3. 钟角蛙（Ornata）
- **外观**：鲜艳绿色 + 红色/橙色花纹 + 高耸眉角
- **物种**：*C. ornata*
- **特点**：比南美角蛙更"华丽"，眉角像小恶魔的角
- **价格**：¥200-500（野生色），¥500-1500（变异色）
- **推荐**：⭐⭐⭐⭐

### 4. 霸王角蛙（Cornuta）
- **外观**：最大体型 + 头顶极长的"角"（眼睑突起）
- **物种**：*C. cornuta*
- **特点**：角蛙之王，体长可达20cm，霸气十足
- **价格**：¥800-3000
- **注意**：对饲养环境要求更高，新手慎入

### 5. 蝴蝶角蛙（Fantasy = C. cranwelli × C. cornuta）
- **外观**：南美角蛙和霸王角蛙的杂交后代，头顶有中等角
- **遗传**：杂交品种（不可育或繁殖力低）
- **特点**：兼具南美角蛙的皮实和霸王角蛙的"角"
- **价格**：¥300-800
- **推荐**：⭐⭐⭐⭐（颜值不错+相对皮实）

### 6. 变异色型（Morphs）

| 色型 | 外观 | 价格范围 |
|------|------|:---:|
| 草莓（Strawberry） | 粉/桃色 | ¥300-800 |
| 蓝玉（Samurai Blue） | 蓝色调 | ¥500-2000 |
| 薄荷（Mint） | 浅绿/薄荷绿 | ¥500-1500 |
| 紫玉（Purple） | 紫色调 | ¥800-3000 |
| 黑化（Melanistic） | 接近全黑 | ¥1000-3000 |
| 铂金（Platinum） | 接近纯白 | ¥1500-5000 |

## 三、选购指南

### 按预算选

| 预算 | 推荐 |
|------|------|
| ¥50-150 | 普通绿角蛙 |
| ¥150-300 | 白化角蛙 |
| ¥300-800 | 蝴蝶角蛙、草莓、钟角蛙 |
| ¥800-2000 | 霸王角蛙、高端色型 |
| ¥2000+ | 铂金、紫玉、顶级色型 |

### 健康角蛙检查清单

- [ ] **身体圆润**（不是扁塌的，不是水肿的）
- [ ] **眼睛明亮**、无浑浊
- [ ] **嘴巴闭合正常**（下颚不下垂）
- [ ] **皮肤湿润**、无破损、无红肿
- [ ] **应激反应存在**（碰它有反应）
- [ ] **无皮下气体**（不是"气鼓鼓像气球"——这是气肿症）

### 幼体 vs 成体

- **幼体（2-4cm）**：价格低，但死亡率相对高
- **亚成体（5-8cm）**：已稳定进食，最推荐新手买
- **成体（10cm+）**：性别可确定，价格较高

角蛙是一个"入手门槛极低，但精品天花板极高"的品种。¥50就能买到一只健康绿角蛙，但也有¥5000+的铂金变异。不管你选择哪个，记住：一只健康的普通角蛙，永远比一只生病的顶级色型更值得带回家。
